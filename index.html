Genetic Algorithms

This project is a simple Java package for writing genetic algorithms (GAs). It came about when I was writing a genetic algorithm that learnt to tile the plane, and was further refined when I used the same framework to evolve game strategies for Prisoners Dilemma. It is not optimised for speed, but rather for ease of use and conceptual simplicity.

Download the distribution, or read the JavaDoc online.

References

Mitchell, Melanie, (1996), An Introduction to Genetic Algorithms, MIT Press.
